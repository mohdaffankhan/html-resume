# Resume HTML Assignment

This project is a resume in a fully semantic HTML and CSS layout while maintaining identical structure, spacing, and readability.

## Setup
Simply open `index.html` in any modern browser.

## Tech Used
- HTML5 (semantic tags)
- CSS3 (layout, spacing, typography)

## Features
- Clean spacing & alignment
- Semantic HTML for accessibility
- Resume-friendly typography
